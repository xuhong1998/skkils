# 日记标签提取系统使用指南

## 概述

本系统为日记skills添加了基于AI的智能标签提取功能，能够在写日记时自动提取标签并添加到日记文件末尾，用于构建Obsidian知识图谱。

## 功能特性

### AI驱动标签提取
- 使用精心设计的提示词让AI理解日记内容
- 提取主要标签、子标签、情绪标签、地点标签、人物标签
- 基于语义理解而非简单的关键词匹配

### Obsidian集成
- 标签块格式符合Obsidian识别标准
- 标签在Obsidian关系图谱中自动显示
- 相同标签的日记自动关联

### 无缝集成
- 集成到现有的三个日记skills中
- 写日记时自动执行标签提取
- 零交互或可选择交互模式

## 标签分类体系

### 主要标签（每个日记3-5个）
- **#投资理财**：基金、股票、黄金、白银、交易、亏损、盈利
- **#数字成瘾**：抖音、短视频、刷、手机、时间浪费
- **#家庭关系**：妈妈、爸爸、家庭、亲情、冲突、沟通

- **#生活日常**：饮食、运动、休息、购物、家务
- **#工作职业**：上班、公司、项目、会议、加班

### 子标签（主要标签的细分）
- **投资理财**：#基金交易、#市场观察、#投资学习、#风险控制
- **#数字成瘾**：#抖音使用、#自控管理、#时间管理
- **#家庭关系**：#家庭冲突、#亲情交流、#社会压力
- **#学习成长**：#阅读完成、#技能学习、#自我提升
- **#生活日常**：#运动健康、#饮食健康、#家政管理

### 情绪标签（2-3个）
- **正向**：#开心、#满足、#成就感、#平静
- **负向**：#焦虑、#烦躁、#压力、#沮丧、#自责
- **中性**：#平静、#思考、#观察

### 地点标签（1-2个）
- 城市：#武汉、#新洲等
- 场所：#家中、#公司、#公园、#餐厅等

### 人物标签（可选）
- 家人：#爸爸、#妈妈、#姑姑等
- 朋友、同事、其他重要人物

## 使用方法

### 1. diary-assistant（详细日记写作）

**触发方式：**
- 「帮我写日记」
- 「记录今天」
- 「写日记」
- 「今天的日记」

**标签提取流程：**
1. 完成启发提问和内容整理
2. 自动触发AI标签提取
3. 显示提取结果供确认
4. 在日记文件末尾添加标签块

**示例输出：**
```
✅ 标签提取完成！

📊 主要标签：#生活日常 #运动健康 #饮食健康 #技术学习
🏷️ 子标签：#散步 #走楼梯 #家政 #Mac #编程
😊 情绪标签：#平静 #反思 #规划
📍 地点标签：#武汉 #家中 #公园

正在添加标签块到日记文件末尾...
✅ 标签块已添加！
```

### 2. diary-quick-append（快速日记追加）

**触发方式：**
- 「写日记：今天做了什么，有什么感受...」
- 「写日记 2-1：今天做了什么...」
- 「写日记 02-01：今天做了什么...」

**标签提取流程：**
1. 自动解析输入并润色内容
2. 后台静默执行AI标签提取
3. 在日记文件末尾添加标签块
4. 完成Git推送

**特点：**
- 零交互，不显示标签提取过程
- 保持快速用户体验
- 自动保存标签

### 3. diary-scatter-organizer（时间戳日记整理）

**触发方式：**
- 「整理日记」
- 「整理记录」
- 「organize diary」

**标签提取流程：**
1. 解析时间戳记录并分类
2. 基于时间段深度分析
3. 提取更丰富的标签（包含健康、情绪等）
4. 在日记文件末尾添加标签块

**增强功能：**
- 时间段标签（上午/下午/晚上）
- 健康信息标签（运动、饮食、情绪）
- 行为模式标签（活动模式和习惯）

### 4. diary-tag-manager（独立标签管理）

**触发方式：**
- 「提取今日标签」
- 「为日记[日期]提取标签」
- 「重新分析今日标签」
- 「批量提取所有标签」
- 「重新分析历史日记标签」

**手动标签提取流程：**
1. 读取指定的日记文件
2. 使用AI提示词提取标签
3. 生成标准格式的标签块
4. 在日记文件末尾追加标签块

**批量处理：**
- 可以为所有历史日记批量提取标签
- 支持重新分析现有标签
- 提供进度显示和错误处理

## 标签块格式

### 标准标签块

所有日记文件末尾都会添加以下格式的标签块：

```markdown
---
## 🏷️ AI提取标签

### 主要标签
#生活日常 #运动健康 #饮食健康

### 子标签  
#散步 #走楼梯 #家政 #健康饮食

### 情绪标签
#平静 #反思 #满足

### 地点标签
#武汉 #家中 #公园

### 人物标签
#爸爸 #朋友

---
```

### 分类说明

- **必须以**`---`**开始和结束**，与日记内容分隔
- **标题必须是**`## 🏷️ AI提取标签`
- **每个分类的标题格式**：`### 分类名`
- **标签之间用空格分隔**，不要用逗号
- **标签必须以**`#`**开头**

## Obsidian知识图谱集成

### 关系图谱显示

标签块中的标签会被Obsidian自动识别：

**节点类型：**
- 📄 日记节点：显示日期和标题
- 🏷️ 标签节点：显示标签名称

**连线关系：**
- 日记节点 ↔ 标签节点（双向连接）
- 相同标签的所有日记自动关联

### 图谱效果示例

```
        #生活日常
         ↗  ↖
    03-01    03-02
         ↘  ↙
      #运动健康
```

## 标签质量保证

### AI提示词优化

- 详细的提取标准和示例
- 语义理解指导
- 格式化输出要求
- 质量评估标准

### 提取原则

**必须提取：**
- 明确提到的投资活动（交易、亏损、学习）
- 明确的数字成瘾行为（刷抖音、看短视频）
- 重要的人际关系事件（家庭冲突、相亲）
- 重要的学习活动（阅读书籍、学习技能）
- 重要的生活事件（特殊聚餐、旅行、运动）

**选择性提取：**
- 一般性描述（根据重要性和频次决定）
- 重复性行为（评估是否值得追踪）
- 具体细节（除非有特殊意义）

**不提取：**
- 过于具体的描述（如具体食物名称）
- 日常琐碎（如"洗脸"、"刷牙"）
- 一时的情绪发泄（除非有重要意义）
- 临时性的小事（如临时决定的小事）

### 质量要求

1. **准确性**：标签必须准确反映日记内容
2. **一致性**：同类型内容使用相同的标签
3. **平衡性**：主要标签和子标签比例合理
4. **实用性**：提取的标签对知识图谱构建有实际价值
5. **简洁性**：标签数量适度，避免过度标记

## 故障排除

### 常见问题

**问题1：标签块格式不正确**
- 检查标签是否以`#`开头
- 确保标签块以`---`开始和结束
- 验证标签之间用空格分隔

**问题2：Obsidian不识别标签**
- 检查标签块格式是否符合标准
- 刷新Obsidian关系图谱
- 重启Obsidian

**问题3：标签提取质量不佳**
- 检查日记内容是否清晰明确
- 调整AI提示词的提取标准
- 手动调整提取的标签

**问题4：标签重复或冗余**
- 检查标签命名规范
- 合并相似的标签
- 清理不必要的标签

### 错误处理

| 错误 | 处理 |
|------|------|
| 日记文件不存在 | 提示用户提供正确的日记路径 |
| 日记内容为空 | 提示用户日记内容为空，无法提取标签 |
| AI提取失败 | 重试一次，失败后提示用户手动添加标签 |
| 标签块已存在 | 询问用户是否覆盖现有标签块 |
| 文件写入失败 | 提示用户检查文件权限 |
