## 核心原则

| 原则 | 说明 |
|------|------|
| **AI驱动** | 使用AI提示词而非Python代码进行标签提取 |
| **语义理解** | 充分利用AI的语义理解能力 |
| **标准化格式** | 标签块格式统一，便于Obsidian识别 |

## 标签分类体系

### 主要标签（每个日记3-5个）
- **#投资理财**：基金、股票、黄金、白银、交易、亏损、盈利
- **#数字成瘾**：抖音、短视频、刷、手机、时间浪费
- **#家庭关系**：妈妈、爸爸、家庭、亲情、冲突、沟通
- **#学习成长**：阅读、学习、思考、计划、反思
- **#生活日常**：饮食、运动、休息、购物、家务
- **#工作职业**：上班、公司、项目、会议、加班

### 子标签（主要标签的细分）
- **投资理财**：#基金交易、#市场观察、#投资学习、#风险控制
- **数字成瘾**：#抖音使用、#自控管理、#时间管理
- **家庭关系**：#家庭冲突、#亲情交流、#社会压力
- **学习成长**：#阅读完成、#技能学习、#自我提升
- **生活日常**：#运动健康、#饮食健康、#家政管理

### 情绪标签
- **正向**：#开心、#满足、#成就感、#平静
- **负向**：#焦虑、#烦躁、#压力、#沮丧、#自责
- **中性**：#平静、#思考、#观察

### 地点标签
- #武汉、#新洲、#公园、#家中、#公司、#餐厅等

### 人物标签
- #爸爸、#妈妈、#姑姑、#朋友、#同事等

## 标签提取提示词

AI标签提取使用以下提示词模板：

**角色设定：**
你是一位专业的日记分析师，擅长从文本中提取有意义的标签，用于构建知识图谱和关联分析。

**任务目标：**
分析用户提供的日记内容，提取符合以下标准的标签，用于构建日记知识图谱。

**提取标准：**
- 主要标签：每个日记提取3-5个，涵盖核心主题
- 子标签：每个主要标签对应2-3个更具体的标签
- 情绪标签：反映日记中的情绪状态（2-3个）
- 地点标签：重要地点（1-2个）
- 人物标签：主要人物（可选）

**标签命名规范：**
- 使用中文标签
- 格式：`#标签名`
- 保持简洁（2-6字为宜）
- 避免过于具体
- 使用标准化的同义词

**输出格式：**
按照以下Markdown格式输出标签块：

```markdown
---
## 🏷️ AI提取标签

### 主要标签
#标签1 #标签2 #标签3

### 子标签  
#子标签1 #子标签2 #子标签3

### 情绪标签
#情绪1 #情绪2

### 地点标签
#地点1 #地点2

### 人物标签
#人物1 #人物2

---
```

**提取质量要求：**
1. 准确性：标签必须准确反映内容
2. 一致性：同类型内容使用相同的标签
3. 平衡性：主要标签和子标签比例合理
4. 实用性：提取的标签对知识图谱构建有实际价值
5. 简洁性：标签数量适度，避免过度标记

## 错误处理

| 错误 | 处理 |
|------|------|
| 日记文件不存在 | 提示用户提供正确的日记路径 |
| 日记内容为空 | 提示用户日记内容为空，无法提取标签 |
| AI提取失败 | 重试一次，失败后提示用户手动添加标签 |
| 标签块已存在 | 询问用户是否覆盖现有标签块 |
| 文件写入失败 | 提示用户检查文件权限 |

## 用户配置

## Obsidian集成

### 关系图谱显示

标签块中的标签会被Obsidian自动识别并在关系图谱中显示：

**节点类型：**
- 📄 日记节点：显示日期和标题
- 🏷️ 标签节点：显示标签名称

**连线关系：**
- 日记节点 ↔ 标签节点（双向连接）
- 相同标签的所有日记会自动关联到同一个标签节点

### 图谱效果

```
        #生活日常
         ↗  ↖
    03-01    03-02
         ↘  ↙
      #运动健康
```

### 配置建议

建议配置Obsidian关系图谱以优化标签显示：

```json
{
  "showTags": true,
  "tagNodeSize": "usage_weighted",
  "colorGroups": [
    {
      "query": "investment",
      "color": "#FF6B6B"
    },
    {
      "query": "digital_addiction",
      "color": "#4ECDC4"
    },
    {
      "query": "family",
      "color": "#45B7D1"
    }
  ]
}
```
