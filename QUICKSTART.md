# 日记标签提取系统 - 快速开始指南

## 🚀 5分钟快速上手

### 第一步：了解标签提取功能

你的日记系统现在支持AI自动提取标签！当你写日记时：

**✅ 自动发生的事情：**
- AI分析你的日记内容
- 提取有意义的标签
- 在日记文件末尾添加标签块
- 在Obsidian中自动创建知识图谱

**📊 提取的标签包括：**
- 主要标签：#投资理财 #数字成瘾 #学习成长
- 子标签：#基金交易 #抖音使用 #阅读完成
- 情绪标签：#平静 #焦虑 #满足
- 地点标签：#武汉 #家中 #公园

### 第二步：使用日记功能触发标签提取

#### 方式1：详细日记写作
```
你：帮我写日记
```
系统会：
1. 引导你完成日记写作
2. **自动提取标签并显示给你**
3. 在日记末尾添加标签块
4. 显示提取结果供确认

#### 方式2：快速日记追加
```
你：写日记：今天去公园散步，心情不错
```
系统会：
1. 快速处理并保存日记
2. **后台静默提取标签**
3. 自动添加标签块
4. 不干扰你的使用体验

#### 方式3：时间戳日记整理
```
你：整理日记：8:30 起床，10:00 学习，12:00 午餐...
```
系统会：
1. 整理时间戳记录
2. **深度分析并提取详细标签**
3. 包含健康和情绪标签
4. 生成完整的结构化日记

### 第三步：在Obsidian中查看效果

#### 打开Obsidian关系图谱
1. 打开Obsidian
2. 点击左侧的"关系图谱"图标
3. 你会看到：
   - 📄 日记节点（日期）
   - 🏷️ 标签节点（标签名称）
   - 🔗 连线（关联关系）

#### 标签关联效果
```
        #生活日常
         ↗  ↖
    03-01    03-02
         ↘  ↙
      #运动健康
```
相同的标签会自动连接相关的日记！

### 第四步：使用独立标签管理功能

#### 手动提取标签
```
你：提取今日标签
```
系统会为今天的日记提取标签。

#### 批量处理历史数据
```
你：批量提取所有标签
```
系统会为所有历史日记（27篇）批量提取标签。

## 💡 最佳实践

### 写日记时的建议

1. **内容清晰明确**：AI更容易提取准确标签
2. **记录重要事件**：重要事件值得追踪
3. **表达情感状态**：情绪标签对知识图谱有价值
4. **记录活动模式**：有助于标签关联分析

### 标签管理建议

1. **定期检查标签**：确保标签质量
2. **合并相似标签**：避免标签冗余
3. **新增标签规范**：保持标签体系一致性
4. **清理无效标签**：删除不常用的标签

## 🆘 遇到问题？

### 常见问题

**Q: 标签提取不准确怎么办？**
A: 可以手动调整标签，系统会学习你的偏好。

**Q: Obsidian不显示标签？**
A: 检查标签块格式，刷新Obsidian关系图谱。

**Q: 标签太多或太少？**
A: 调整AI提示词的提取标准，或手动增删标签。

**Q: 想查看所有标签统计？**
A: 使用"查看标签统计"命令，或查看 `README_TAGS.md`。
